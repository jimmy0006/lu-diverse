이 프로젝트는 WebGL로 빌드한 게임을 업로드하고, 서로 플레이 할 수 있는 프로젝트야.
게임은 AWS S3에 업로드 할거고, S3에 접속해서 게임을 불러오는 방식으로 만들거야.

메인 페이지(로그인 전)
- 로그인 버튼
- 게임 검색
- 인기있는 게임 리스트 (리스트로 표시됨)

메인 페이지(로그인 후후)
- 로그아웃 버튼
- 게임 검색
- 인기있는 게임 리스트(리스트로 표시됨)
- 게임 업로드 버튼
- 내가 찜한 게임 리스트(새로운 페이지로 이동 버튼)
- 내가 올린 게임

게임 선택 페이지
다음의 항목이 리스트로 표현됨
- 게임 이미지
- 게임 제목
- 간단한 설명
- 버전
- 게임 조회 수
- 게임 게임 찜한 수

게임 선택 시
- 게임 플레이(테두리는 lu-diverse 틀에 넣고 안에 frame으로 webGL 내장)
- 게임 바로 아래에는 조회수, 찜한 수 (찜하기 버튼)
- 더 아래쪽에는 게임 버전 이력

내가 올린 게임
- 게임 업데이트
- 업로드 시 버전 업그레이드 (버전은 기입할 수 있으나, 기입하지 않으면 마이너 버전을 1 올리는 방식)
- 게임 사진 및 게임 파일 업로드(게임 파일은 1GB가 최대 용량, zip 형식만 지원)
- 게임 설명 업데이트 가능(건들지 않으면 기존 설명 유지)
- 게임 업데이트 시 조회 수와 찜한 게임 수는 초기화되지 않음

게임 업로드
- 입력값(게임 제목, 설명, 게임 업로드(1GB 미만의 zip 파일만))
- 버전은 1.0.0이 기본값이지만, 개발자가 원할 시 수정 가능

내가 찜한 게임 리스트
- 찜한 게임 리스트(버전, 업로드 날짜, 업데이트 날짜, 조회 수, 찜 수)
- 게임별로 내가 플레이한 시간
